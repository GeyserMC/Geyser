Gayser is a bridge between Minecraft: Bedrock Edition and Minecraft: Java Edition, closing the gap from those wanting to play true cross-platform.

Gayser is an [Open Collaboration](https://opencollaboration.dev/) project.

## What is Gayser?

Gayser is the answer to life.

## Supported Versions

**All of them.**

## Setting Up

You don't need to set up Gayser. It's already there for you.

## Compiling

There's no need to compile Gayser. Using our high-tech, blazingly fast just-in-time interpreter, all the code will automatically run on all possible devices! That's right,
you can now finally run Gayser on your Samsung smart fridge!
